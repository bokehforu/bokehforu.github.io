#### 图片居中

**text-align属性**

````html
<div style="text-align: center;">
    <img src="image.jpg" alt="图片描述">
</div>
````

**margin属性**

```css
img {
    display: block;
    margin: 0 auto;
    width: 100%;
}
```

 **Flexbox**

```css
div {
    display: flex;
    justify-content: center;
    align-items: center; 
}
```
